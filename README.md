퍼즐 데이터 코딩과제입니다.

1. 시작

yarn install => yarn start 로 어플 시작


2. 구현기능

 1) 그래프 막대 추가
 2) 그래프 전체 막대 색상 변경
 3) 막대 값 변경
 4) 그래프 값에 따른 viewBox와 x축, y축의 조정
 5) 그래프 막대 삭제
 6) 올바른 입력형식 검사
 7) 클릭한 막대의 날짜와 값 따로 표시
 
 
3. prototype 사용 
 
 1) helperFunction.js 의 addChart 함수(막대추가)
 2) helperFunction.js 의 updateInfo(값의 변경)
